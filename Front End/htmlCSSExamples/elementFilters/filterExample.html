Filter Examples

Below are examples of all the filter types in CSS3.

Remember to use the webkit prefix so that they are viewable in Chrome

Greyscale

Makes the image more grey. A value of 1 is max.

Blur

Adds blur to an image. The value is in pixels.

Sepia

Creates a sepia effect. A value of 1 is max.

Hue Rotate

Adds a hue to the image. The value is in degrees.

Saturate

Saturises the image. The value of 1 is max.

Opacity

Changes the opacity of the image. 0 is invisible, 1 is visible.

Brightness

Changes the brightness of the image. The value of 1 is max.

Contrast

Changes the contrast of the image. The value of 1 is max.
