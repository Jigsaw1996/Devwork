Display Examples

Block Level

Block level elements, such as paragraphs and divs, take up their full line, only allowing one element as part of it's row of content.

Example:

Despite only taking up 40% of the container, I take up my full row. Any further content will be on a new row.

As you can see, I'm on a new line.

Inline Level

Inline level elements only take up the row space that they need to. Inline level elements include anchor tags, strong tags and spans.

Inline elements can be placed within block level elements, but this can't be done vice versa. They can't be resized.

Example:

The line doesn't cut off after this anchor tag, because it is inline and only uses the line space necessary.

The same goes for strong, Emphasis and spans (used for styling parts of text elements).

Inline-Block Level

Inline-Block level elements are a mix of the two other levels above. They only take up necessary line spacing, but can have their width/height changed. An example of this is the image tag.
