Effect Examples

Whilst CSS has animation posibiliies, JQuery supports additional options right out the box.

Using these alongside events can create some cool opportunities for websites.

Syntax

To declare a toggle to show/hide a p element, do the below.

$("p").toggle();

Some effects take values, such as a number argument for fades and slide which dictates how long the effect takes in miliseconds.

Examples

Note that fade to is used to make an element semi-transparent rather than invisible.

Effect Types

  • .show(); - Shows a previously hidden element.
  • .hide(); - Hides a previously visible element.
  • .toggle(); - Toggles the visibility, from visible to not visible.
  • .slideDown(); - Slides down a hidden element.
  • .slideUp(); - Sides up a visible element.
  • .slideToggle(); - Toggles slides down and slides up.
  • .fadeIn(); - Fades in a hidden element.
  • .fadeOut(); - Fades out a visible element.
  • .fadeToggle(); - Toggles fade ins and outs.
