Selector Examples

In JQuery, you can easily select HTML elements.

The syntax for this is below.

var anchors = $("a");

Every anchor tag on the page is now linked to the anchors variable.

Examples

  • Class Selector makes this red
  • ID Selector makes this blue

Note that, rather than changing the styles through inline styling, we instead make JQuery add a class to it.

To take a class away, do .removeClass(class);

Psuedo Selectors

Below are a list of some of the psuedo selectors available.

  • $("a:first"); // Selects the first anchor tag on the page
  • $("a:last"); // Selects the last anchor tag on the page
  • $(":header"); // Selects the header elements of the page (h1, h2 etc)
  • $(":animated"); // Selects all elements in the progress of animation upon the time of run.
  • $(":checked"); // Selects all elements which are checked or selected.
  • $(":even"); // Selects all even elements
  • $(":focus"); // Selects element which is currently focused

For example, the :first psuedo selector has been used on the first list item.
