Object Instance Examples

When creating objects, we may need to create multiple objects of the same type.

To do this, we can set up a constructor which we can create the objects from.

Building a Constructor

To build a constructor, we create the necessary keys and values within a function, using the "this" keyword so that they apply to the object created.

For example:

var hero = function(firstName, lastName, title) {
firstName: firstName,
lastName: lastName,
title: title
};

This function will create an instance.

Creating an Instance

Once we have create a constructor, we can start creating instances.

We do this using the "new" keyword and providing the correct key values.

var batman = new hero("bruce", "wayne", "batman");

Adding Functions

Because constructors are used to create values, there's little room for functions.

We can instead use prototypes.

We do this like below:

Hero.prototype.getName = function() {
return this.firstName + " " + this.secondName;
};

This will add the function to the constructor and all children.
