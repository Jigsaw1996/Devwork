Function Examples

Functions are common in programming languages. They allow for code to be reused multiple times and, in many cases, for multiple purposes.

They are first defined, and then called with relevant arguments. They can be executed as many times as necessary.

Syntax

First, a function must be defined. This involved writing relevant code which will run when the function is called.

Any arguments should be placed in the brackets, and within the function where necessary. The values are provided when called.

To define the function:

function functionName (arguments) {
relevant code
}

After it has been defined, we can call it as many times as we like, such as below.

functionName(argumentValues);

The argument values will then replace the argument names placed within the definition.

Callbacks

Callbacks involve running a function through another function.

This is common in programming, and is useful for making the most use of functions.

Below is the code you'd typically see from this:

function wotDayIsIt(callback){
var day = (new Date().getDay());
callback(day)
}

var dayOfWeek = function(day){
if (day === 0 || day === 6) {
console.log("it’s the weekend, yippee !");
}
else {
console.log("bummer, it’s a work day")
}
}

wotDayIsIt(dayOfWeek);

In this example, we define a function with one argument, which we will be the name of the function we wish to run through it.

In this case, we use the wotDayIsIt function to get the day, and the day is then ran through the dayOfWeek function to decide it's message.
