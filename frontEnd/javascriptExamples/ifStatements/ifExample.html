If Statement Examples

If statements are used to only execute code if certain conditions are met.

For instance:

  • If 1 + 1 = 2, make this green.
  • If 2 + 2 = 5, make this red.

View the JS file linked to see one in action.

Syntax

JS If Statements are written like below.

if (operators) {
statements
}

If the operator evaluates to true, the statements contained will be read by the browser.

Operators

Operators are used to output a Boolean value. If the Boolean is True, if statements will execute.

Here are the operators.

  • == - Equal to
  • === - Equal to and of the same type
  • != - Not equal to
  • !=== - Not equal to or not of the same type
  • > - Greater than
  • < - Less than
  • >= - Greater than or equal to
  • <= - Less than or equal to

Brackets can be used to make the calculations more complex, just like normal maths.

For instance:

<(2 +3) == (1 + 4)

This evaluates to True, because it calculates the brackets first.

Logical Operators

Following on from this, there are further operators which can help us with achieving a Boolean value.

These are listed below.

  • && - Returns true if both operands are true.
  • || - Returns true if one of the operands is true.
  • ! - Returns true if the operand is false, and false if the operand is true.

Using these with brackets holding the calculations on the left and right will check both and provide the relevant response.

If Else and Else

An if statement can be accompanied by two other satements that decide what happened

The "if else" statement sets another operator which will be checked if the first returns False. There's no limit to the amount which can be added.

Finally the "else" statement, placed last, is used to define what should happen if all previous checks returned false. It doesn't require an operator.

Full example:

if (operators) {
statements
}
if else (operators){
statements
}
else {
statements
}
