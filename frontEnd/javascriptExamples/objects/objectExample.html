Object Examples

Objects are used to create items, people or other object types and relevant data types.

Contained data works in key:data pairs. The data can be a data type, a function or an array.

Syntax

Objects are defined using a variable, with the value being contained in curly braces.

The contained data is then given a key:data pair.

var objectName = {
containedVariable: "Value";
containedFunction: function() {
code...
}
containedArray:
}

Note that we use anonymous functions when declared from within an object.

To access a object item, we use objectName.keyName. This will return the key's value.
