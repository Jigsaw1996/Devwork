Loop Examples

Loops are used to run the same statements repeatedly as long as a condition is true.

For Loops

For loops take three arguments, listed below.

  • Initial expression, used to create a variable relevant to the loop.
  • Checked before each run, if False it stops running.
  • Increment Expression, this action is completed at the end of each loop.

for ([initialExpression]; [condition]; [incrementExpression]) {
statement
}

For an example, view the console. We have used a loop to print the numbers one through five.

Looping Arrays

Arrays store multiple data and data types. What if we wish to make changes to all of them?

We can use a for loop to accomplish this, as seen below.

for (var num in myArray) {
console.log(myArray[num]);
}

What this is doing is creating a new variable, num, which will represent the current array index number.

It will cycle through all indexes and output each array item to the console individually.

Once this for loop has cycled through all list items, it automatically stops.
