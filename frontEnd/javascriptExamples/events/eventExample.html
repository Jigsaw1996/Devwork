Event Examples

In JQuery, we can use events to impact user experience.

Events are triggers which users execute when using the site.

They can be clicks, hovers, clicking a button etc.

They are executed like the below.

$("p").on("click", function(){
// Relevant code executed on click
})

Examples

Click me, I turn blue

Hover over me, I turn green

When you scroll, I turn red

As you can see, we have made use of adding and removing classes, rather than adding inline styling. This is best practice.

Event Types

  • .click() - Activated when the user clicks.
  • .keydown() - When the user holds a key
  • .keypress() - When the user presses a key
  • .keyup() - When the key rises
  • .mouseover() - Whilst the mouse is over an element
  • .mouseout() - When the mouse moves off an element
  • .mouseenter() - When the mouse enters an element
  • .mouseleave() - When the mouse leaves an element
  • .scroll() - When the user scrolls on the element
  • .focus() - When the user focuses on the element
  • .blur() - When the element is blured
  • .resize() - When the element is resized
