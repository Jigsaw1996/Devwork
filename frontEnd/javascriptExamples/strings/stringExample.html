String Examples

Strings are a common data type in Javascript, they store text within a variable.

The browser knows a string by the speech marks surrounding it, these can be double or single quotes.

Below is an example of a string stored within a variable.

var x = "Lorem Ipsum";

Strings can be added together, as seen below.

var x = "lorem ";
var y = "ipsum";
var z = x + y;

In the above example, z will equal "lorem ipsum" as it adds the two together.

Take note of the space after lorem. Remember to add spaces between words.

Characters

What if we want speech marks or other characters as part of the string? Surely this will end the string prematurely?

The solution is to use the other type of speech mark to the one you used to declare it.

For Example:

var x = "This string contains 'speech marks' as part of the text."

The speech marks are now saved as part of the string.

String Methods

There are a number of methods which we can use on strings.

Here are some of these.

  • charAt() - Returns the character which is at the index provided as an argument.
  • charCodeAt() - Returns the unicode of the character at the index provided.
  • concat() - Adds a string to the end of the other.
  • includes() - Checks if the string contains a character or string.
  • startsWith() - Checks if the string begins with a character or string.
  • endsWith() - Checks if the string ends with a character or string.
  • indexOf() - Returns the first index value of a provided character.
  • lastIndexOf() - Returns the last index value of a provided character.
  • search() - Searches the string for a certain character, string or regular expression and returns it's value.
  • toUpperCase() - Converts the text to uppercase.
  • toLowerCase() - Converts the text to lowercase.

For example, we will use the below to make the string uppercase, and then check if it ends with an uppercase G.

var green = document.getElementById("green");

var x = "lowercase string";

>var y = x.toUpperCase();

var z = y.endsWith("G");

if (z) {
green.style.color = "green";
}

>else {
green.style.red = "red";
}

If it does, this text will be green.
