Variable Examples

Variables are at the core of a programming language.

They are used to store data which the program will use.

Once defined, they can be redefined, changed or used elsewhere.

Declaring Variables

To declare a variable in JS, use the "var" keyword, followed by the name, then assign the value.

The equals sign is the assign key in JS.

var x = 1;

This gives the variable x a value of 1.

In this case, the data type used is a number, there are more data types which will be covered below.

Include a semi-colon at the end of each operation to avoid error.

Calling Variables

Once a variable has been defined, we can call it's value later.

See the below:

var x = 1;

var y = x

y will have a value of 1, because when declaring it's value, we placed the name of the x variable. This calls it's value when read.

Data Types

In the above examples, the values of variables have been numbers. Naturally, there are more than just numbers used by applications.

There are several data types we should look at.

  • Booleans - True or False value.
  • Numbers - A numeric value, either whole or with a floating point.
  • Strings - Text, a value surrounded with speech marks.
  • undefined - When no value has been given yet, it has this as it's data type.
  • null - A special keyword, more on this later.

Below are examples of the big three.

var boolean = True;

var number = 3.14;

var string = "This is some text";

Type Coercion

If you try and add different types of data together, JS will try to perform these.

For numbers and text, it can add them together and output a string, like below.

x = "The answer is " + 42 // "The answer is 42"

y = 42 + " is the answer" // "42 is the answer"

JS adds these together.

Any other circumstance and it wont have the intended result.
