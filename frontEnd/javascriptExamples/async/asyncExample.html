Async Examples

Async Programming means taking functions etc outside of the standard load, and places it into a queue only loaded after all other elements.

This is useful so as to make sure everything vital is loaded first.

A useful way of making sure all html has loaded before JS loads, wrap the js files in the following:

$(document).ready(function() {
// Relevant Code )};
