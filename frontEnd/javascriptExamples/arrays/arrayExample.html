Array Examples

Arrays are used to store data and multiple data types within a single variable.

This means that, rather than creating hundreds of variables, you only need one to manage your list.

Arrays are indexed, and can be accessed through a zero-based index system.

Syntax

Below is how you declare a variable.

var myArray = [1, 3, 3.14, "string", false];

As you can see above, we use braces to hold the data, each one being comma seperated. Data type rules still apply, so speech marks are still needed for strings etc.

There's no limit to the amount of array items you can add, though try and keep the lists maintainable.

Accessing Array Items

So how do we access data when it's stored in an array?

We do this using the variable, curly braces and the index of the item we're accessing.

To accessing the second array item, do the below.

var x = myArray[1];

This assigns the variable x a value of 3, because that's the second value in the array.

Note that array indexes are zero-based. The first item has an index of 0, the second of 1 and so on.

Array Length

If we need to find out the length of an array, that being how many items an array contains, there is a property we can use.

Using .length on the variable assigned to the list returns the length of the list, like below.

var arrayLength = myArray.length;

The above assigns the arrayLength variable a value of 5, because that's how many items the myArray array contains.

Array Methods

There are a number of methods which can be used on arrays.

  • push() - The argument will be added to the end of the array as a new index.
  • unshift() - Adds an element to the front of the array.
  • shift() - Removes the first index of the array and returns it.
  • pop() - Removes the last index of the array and returns it.
  • indexOf() - Searches the array for a value given as an argument, and returns it's index. If not in the array, it returns -1.

So, for example, the below will add an "additional item" string to the end of the array.

myArray.push("additional item");

View console to see all of the items, including this additional item, printed.
