Media Objects and Panels Examples

Below are examples of two types of containers in Bootstrap.

The first are media objects, which are used to display left-right media content.

Lastly there are panels, which are used to display title-content pairs.

Media Objects

Media objects display content such as images and text on the same row.

This is accomplished with a .media class container. Two divs are then used to contain the left-right content.

The first div should have the .media-left class, the second should have the .media-body class.

The image etc should then be placed within the first, and the content within the second, as seen below.

Nested Media Objects

Media Objects can be nested within one another by making them part of a list.

First, contain the media items within an unordered list with the .media-list class. The list items are given the .media class. Placing a ul within a ul creates the desired effect, as seen below.

Panels

Panels stick your content in a box. This can consist of a title-content pair to make it look a bit more snazzy.

This is done using a div element with the .panel class. Also add .panel-default for default styling.

This contains two divs. The first has a class of .panel-heading, the second has a class of .panel-body.

You can also have other sections, such as a footer using the .panel-footer class.
