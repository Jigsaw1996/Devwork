Wells and Responsive Embed Examples

Bootstrap has two other components which are useful for displaying specific content. Wells are great for displaying small pieces of important text.

Responsive Embeds help display embeded videos etc within a container that scales via aspect ratio.

Wells

Used on text, wells give it an inset effect.

This is done by containing the text within a div, with the .well class.

Responsive Embed

Bootstrap can embed videos, iframes etc with dynamic widths.

This is done by setting a ratio for the video etc to use.

To do this, place the video, embed etc within a div, using the .embed-responsive class. A second class is then needed, depending on the desired aspect ratio.

For 16:9, use .embed-responsive-16by9

For 4:3, use .embed-responsive-4by3

Lastly give the item inside the container a class of .embed-responsive-item

16:3

4:3

You don't need to set frameborder="" when doing this either, Bootstrap sorts this for us.
