Bootstrap Example

Bootstrap is an extention of CSS which gives a bunch of default display possibilies.

It is open-source, free and mobile responsive.

This page is built with Bootstrap, lets explain how it works.

Grid System

Bootstrap displays content using a grid system, which divides each row into 12 columns. Page elements can take up any number of these.

To set this up, firstly we need to set up the grid container using the .container class on the container.

We then need another container for the individual row called .row. This row contains the 12 columns.

Individual elements in this grid then need a class. This class depends on the size of the screen.

Responsive Classes

Column widths of elements within a row are decided by a class, which looks like the below:

.col-xx-12

12 represents the maximum width. If you want it to take up half the screen width, use 6 etc.

The xx should be replaced with the screen size which you want this width to apply to.

The screen sizes are below:

  • xs - Small screens, less than 768px
  • sm - Tablets, equal or bigger than 768px
  • md - Smaller desktops, equal or bigger than 992px
  • lg - Larger desktops, equal or bigger than 1200px

Apply multiple classes to the same column element, so that it has multiple widths depending on screen size.

NOTE: Bootstrap is mobile responsive, so widths on smaller screens will apply to larger screens unless overwritten.

Hidden and Offset

As well as width, we can define the offset of the element at different screen sizes. This is done by applying the below to the element:

.col-xx-offset-3

This will make the column skip 3 columns on the screen size and higher specified in place of the xx. This is useful for centering elements with less than full widths etc.

Similarly we can make it invisible entirely on set screen sizes using the below:

.hidden-xx

The item will then not display at all at that screen size.

Container Widths

By default, containers have left and right margins. However, what about when we want full width elements?

Newer Bootstrap versions have a class for this, as seen below:

.container-fluid

This will make the container stretch to the ends of the screen. This may not look great, so think before deploying.
