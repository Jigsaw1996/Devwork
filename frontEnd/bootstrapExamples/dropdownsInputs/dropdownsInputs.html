Bootstrap Buttons and Inputs

Two prominent components in Bootstrap are buttons and inputs, which have multiple options to achieve different results.

Note that these involve HTML forms and Bootstrap buttons. Please see the respective examples for more info on these.

Dropdowns

Dropdowns used to make the user pick one of multiple options.

First, give a containing div a class of .dropdown.

Then create a button, including the class .dropdown-toggle and a unique ID.

Lastly create a list with a class of .dropdown-menu.

Button Grouping

We've learned buttons and we've learned dropdowns, but what if we want multiple of these on the same row?

Giving the buttons a container with the .btn-group does the trick.

Dropdowns must have their own nested container with .btn-group if they are displayed on the same line.

Inputs

Bootstrap allows us to make changes to the way that inputs display.

To add Bootstrap elements to forms, wrap the input and other elements in a container with an .input-group class.

Add a class of .form-control to the input, and place a span either before or after the input, containing the necessary content.

Lastly, give the span a class of .input-group-addon and an id of #basic-addonx with x as the correct number, from 1 to 3 depending on size.
