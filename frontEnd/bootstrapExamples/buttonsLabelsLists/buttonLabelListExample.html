Bootstrap Buttons, Labels and Lists

Three basic Bootstrap components are buttons, labels and lists. All of these are an important part of UX.

Here, we will explain how to use each of these.

Buttons

As well as page layout, we can also use Bootstrap to create buttons.

Here are some of the buttons we can make.

Default Primary Info Success Warning Danger

There's also a normal link button too.

Labels

Labels are similar to buttons, but they aren't supposed to be clicked. They are instead supposed to provide info etc, typically on headings.

They share the same names and colours as buttons, so they are easy to remember.

These are created using a spans, the .label class and then .label-default or whatever the correct label is.

Lists

We can also create lists using Bootstrap.

This is done using the .lists class on a container and a .list-group class on the ul.

Individual list items should have the .list-group-item class.

Here's the default display example below:

  • List Item 1
  • List Item 2
  • List Item 3
  • List Item 4

To disable a list item, add the .disabled class to it like the fourth item.

To make a list item active, add the .active class.

If you wish to use buttons instead of lists, simply use buttons instead of the list items. The ul isn't needed either, obviously.
