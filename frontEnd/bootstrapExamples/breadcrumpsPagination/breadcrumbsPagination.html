Bootstrap Breadcrumps and Pagination

Bootstrap has some useful page and navigation related elements which help users find their way around.

Two of these are breadcrumbs and pagination, which we will look at here.

Breadcrumbs

Breadcrumbs are typically displayed at the top of the page, as a way of letting users know the page heirarchy, allowing them to return to less specific pages.

This is great for giving users full control over navigation and find content specific to them.

They are created using an ordered list with the .breadcrumb class. List items and links are then added, which represents the crumbs.

The .active class should be added to the page the user is currently on.

Pagination

Pagination displays links to previous and upcoming pages, either by number or name. These are often seen on search engines etc.

This is done usling an unordered list with the .pagination class.

The list items start and end with the double arrow sybmow.

The ones in between are simply the numbers/names relevant.

The pagination should be contained within a nav tag, representing it's status as a navigation element.

The previous page list item has been given the .disabled class, as there's no previous page. The first item has been given the .active class to show it's the current page selected.

The .pagination-lg class can be added to make it larger.

Pager

Similar to pagination, a pager displays links to other pages. However it only has two links, usually containing text.

Just like pagination, it takes an unordered list class. Add the .pager class to it and add the two list items with anchor tags.

  • « Previous
  • Next »

Add the .previous and .next classes to the relevant list item, that way thye align to the left and right hand sides.

  • « Previous
  • Next »

The .disabled class can also be added to a list item if there is no next or previous item.
