Bootstrap Jumbotron and Thumbnail Examples

Bootstrap provides two useful display features for pages. Jumbotron, useful for page headers, and thumbnails, great for containing pictures and similar adjascent content.

Jumbotron

Extends the viewport to hold key information, such as a banner or title for the page. Typically seen at the top of the page.

To make full width, place outside of any .container class elements, instead placing a .container class element within to hold content.

Placing within a container will make it scale appropriately, and give it rounded corners.

For the jumbotron itself, create a container with the .jumbotron class.

Thumbnails

Thumbnails are designed to hold images or text in minmal space within adjascent containers. Bootstrap achieves this with a lot amount of markup.

On a row, set up your grid system with the necessary columns.

Within the columns, add an anchor tag with the .thumbnail class. The content within this anchor tag will be the thumbnail content.
