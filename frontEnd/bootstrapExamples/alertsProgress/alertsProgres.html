Bootstrap Alerts and Progress Bar Examples

Bootstrap offers two useful components for user feedback and updates.

These take the form of alerts and progress bars. Alerts convey information to the user based on their actions and inputs, and progress bars give an idea of how much of a task is complete.

Alerts

Alerts display important information to the user based on their/our actions.

These can take multiple forms, depending on the type of alert we want to give.

Create a div with the .alert class, followed by the .alert-success or equivalent.

Success Alert! Congratulations!
Info Alert! Be Aware of This Information!
Warning Alert! Correct Or Be Wary of This Message!
Danger Alert! Urgent Action Needed!

But what if we want the user to be able to dismiss the message?

Here we need a button added inside the alert. This needs the data-dismiss="alert" attribute (to make sure it's compatible with all browsers) and a class of .close.

The content of this should be a time symbol.

Success Alert! Congratulations!

With this in place, the button will then disappear when clicked.

Naturally this requires Boostrap's Javascript files to work.

You may also want a link in the alert. This requires the .alert-link class for styling purposes.

Progress Bars

Progress bars represent completion of a task by either the user or the site.

They are made using a div container with the .progress class.

Another div is then placed inside this with the .progress-bar class. Use the style="" atribute to assign a width value in percentages.

This width represents the progress.

Place the value in text within this div to display the percentage to the user.

60%

To ensure the text is visable even in low percentages, add a min-width as well, preferably with a value of 2em, to ensure there is always enough room.

To make things look a bit more interesting, we can use .progress-bar-success and other variations on the progress bar. These are the same as buttons, badges etc, so the names should be familiar.

Success
Info
Warning
Danger

We can also add funky strips to it by also adding the .progress-bar-striped class after the colour type.

60%

To animate it, add the .active class as well.

50%

Multiple colours can be added to one progress bar by simply adding multiple progress bars within the container.
