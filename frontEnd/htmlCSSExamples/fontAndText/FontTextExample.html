Font and Text Examples

A lot can be done with fonts and text with HTML and CSS. We will cover examples of these here.

Figure and Figcaption

Applying text to images as captions, titles etc is made easy because of the figure tag.

This is great for the semantic web, as it specifies what text relates to the image.

Font Family

Fonts are given a family that dictates it's appearance.

Some are installed into the browser, others will have to be downloaded.
